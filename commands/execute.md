Load the execute skill, then execute the plan: $ARGUMENTS

Follow the execute skill process:
1. Read and review the plan critically — raise concerns before starting
2. Create task list
3. Group tasks into batches by logical cohesion (not a fixed count)
4. Present batch grouping and explain logic
5. Execute each batch, run verifications, report what was implemented
6. Wait for feedback between batches
7. After all tasks: run full test suite, run build, report completion
8. Stop and ask when blocked — don't guess
